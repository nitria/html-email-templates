Introduction

Christmas is a responsive html email template for e-shops. Knowledge of html and css is required for creating tables and styling them. Tested with litmus( https://www.litmus.com/ ) and campaignmonitor (https://www.campaignmonitor.com/css).

Folders and files that are included:

  • - index.html
  • - christmas.psd
  • - documentation(folder)
  • - images(folder)

How to send Email

GMAIL & YAHOO

1. Extract the nikolas.zip file.

2. Open index.html with your code editor and copy all html code.

3. Go to your gmail and compose a new mail.

4. In the message section right click and press inspect.

5. At the selected div in DevTools right click and select edit as html.

6. Paste the copied html code from index.html and click anywhere on the empty space to show your email

7. Add recipients and subject and send your email. That's it your done!

OUTLOOK

1. Extract the nikolas.zip file.

2. Press New Email

3. Press attach file(papper-clip icon).

4. Choose the index.html file and click the arrow on the insert button, select Insert as Text.

5. Add recipients and subject and send your email. That's it your done!

Customize

If using hex code for colors must be six character long e.g(#000000) not (#000). RGB is supported in most clients but RGBA not, so choose wisely.

Each section starts with table element:

            
              <table>
                <tr>
                  <td>
                  Content here
                  </td>
                </tr>
              </table>

Besides adding width and height in styles always add width and height attributes without units in your tables and td's :

            
              <table style="width: 600px;" width="600">
                <tr>
                  <td style="width: 300px; height: 150px" width="300" height="150">
                  Content here
                  </td>
                </tr>
              </table>

Some outlook clients have rendering issues so use the snippet below to fix them:

Outlook doesnt support css background images so use vml(vector markup language) code

            
              <v:image
                  xmlns:v="urn:schemas-microsoft-com:vml"
                  fill="true"
                  stroke="false"
                  style="
                    border: 0;
                    display: inline-block;
                    width: 450pt;
                    height: 540pt;
                  "
                  src="https://i.ibb.co/7QdM8zs/Merry-Christmas.jpg"
                />
                <v:roundrect
                  xmlns:v="urn:schemas-microsoft-com:vml"
                  xmlns:w="urn:schemas-microsoft-com:office:word"
                  href="#"
                  style="
                    height: 36px;
                    v-text-anchor: middle;
                    width: 130px;
                    position: absolute;
                    left: 230px;
                    top: 500px;
                  "
                  arcsize="1%"
                  stroke="f"
                  fillcolor="#ffffff"
                  >
                  <w:anchorlock />
                  <center
                    style="
                      color: #e74064;
                      font-family: Helvetica, Arial, sans-serif;
                      font-size: 18px;
                    "
                    >
                    Shop Now
                    </center>
                    </v:roundrect>
